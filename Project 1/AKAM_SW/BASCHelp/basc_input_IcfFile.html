(*.icf)

This file contains the image coordinate measurements. All measurements are assumed to be in photo coordinate system of the frame camera. The file should contain the data in the following format:

  • Photo id: photo in which the point is measured. It cannot contain any white space within its string (see example below).
  • Point id: It cannot contain any white space within its string.
  • x: x photo coordinate of the point.
  • y: y photo coordinate of the point.
  • 2x2 weight matrix for image coordinate measurements.
