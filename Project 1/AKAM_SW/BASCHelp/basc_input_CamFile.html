( *.cam)

This file contains information about the involved cameras in the adjustment. For each camera used in the project, the following information should be furnished:

  • Camera ID: camera identification. It cannot contain any white space within its string (see example below).
  • Camera type: FRAME. BASC only takes cameras of type FRAME.
  • xp, yp, c: Camera interior orientation (IO) parameters.
  • 3x3 dispersion matrix of xp, yp and c.
  • Number of camera distortion parameters, nodist. In BASC nodist should be 7 as defined by the SMAC model and the covariance matrix should also be of size 7x7.
    • Camera distortion parameters if nodist > 0
    • Distortion parameters covariance matrix if nodist > 0 (nodist x nodist matrix).
  • GPS antenna offset from the perspective center (dx, dy, dz)
  • GPS antenna offset dispersion matrix (3 x 3 matrix)
