(*.gcp)

This file contains the data for the ground control and tie points along with their covariance (3x3 matrix). Keep in mind the values of minCov1 and minCov2 (specified in the project file) when designating your covariance matrices. The format is:

  •  Point ID: It cannot contain any white space within its string (see example below).
  •  X, Y, Z: the ground coordinate of the point
  •  3x3 dispersion matrix: the dispersion of the object point coordinates

Example:

! In Project File minCov = 1e+4 and minCov2 = 1e-7
!
!
!
!Point ID X Y Z Dispersion

3 0.25e-10 0.0 0.0 0.0 0.25e-10 0.0 0.0 0.0 0.25e-10
