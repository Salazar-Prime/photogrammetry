(*.gps)

This file contains the GPS/INS observation and their dispersion versus time. The data should be organized in the following format:

  • GPS time: time at which this reading was taken and should match the time associated with one of the images in the (*.ori) file.
  • Rotation angles: omega, phi, kappa in degrees obtained from the INS for this time. If INS is not available, just input zeros to maintain the format.
  • Dispersion matrix of the rotation angles: 3x3 matrix. If INS is not available, just input zeros to maintain the format.
  • Position of perspective center: (Xo, Yo, Zo). Also, if only INS is available, then input zeros for Xo, Yo, Zo to maintain the format.
  • Dispersion of Xo, Yo, Zo: 3x3 matrix. If GPS is not available, just input zeros to maintain the format.
