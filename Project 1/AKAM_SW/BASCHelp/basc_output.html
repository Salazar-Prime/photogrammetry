Graphics plot files

BMP files that contain a graphical representation of the normal system and variance-correlation function used for visual inspection.

  • *_FinalNC.bmp: A side-by-side visual representation of the Normal matrix and the Normal vector.
  • *_FinalCorrelation.bmp: A visual representation of the final correlation matrix between the parameters. The diagonal correlation matrix represents an ideal result where no correlation exists between the parameters. Off diagonal elements will show in case the correlation between two (or more) parameters exceeds certain pre-set threshold (e.g., 0.95).
